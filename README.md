# Конфиги для Linux утилит

## Niri/Hyprland

```
curl -fsSL https://install.danklinux.com | sh
```
## Neovim
Конфиг располагается в директории *~/nvim/* в виде init.lua, также требуется установить git clone https://github.com/nvim-treesitter/nvim-treesitter.git ~/.local/share/nvim/site/pack/plugins/start/nvim-treesitter и git clone https://github.com/folke/tokyonight.nvim.git ~/.local/share/nvim/site/pack/plugins/start/tokyonight

## Tmux
Конфиг располагается в директории *~/tmux/* в виде tmux.conf, а также требует установку плагинов с помощью команды mkdir -p ~/.config/tmux/plugins && git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm/
