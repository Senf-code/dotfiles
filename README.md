# Конфиги для Linux утилит

## Neovim
Конфиг располагается в директории *~/nvim/* в виде init.vim

## Tmux
Конфиг располагается в директории *~/tmux/* в виде tmux.conf, а также требует установку плагинов с помощью команды mkdir -p ~/.config/tmux/plugins && git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm/
